### 📝 **프로젝트 상세**
- **개발 기간**
    - 2024.01.25 ~ 2024.02.21 (4주)
- **멤버**
    - Front-End 1
- **기능**
    - 모바일 뷰
    - 사용자 인증: 이메일을 통한 회원가입 및 로그인, 로그아웃
    - 기도 제목 관리: 사용자들이 기도, 감사 제목을 게시하고 수정, 삭제
    - 익명성 보장: 사용자가 부담 없이 참여할 수 있도록 익명성 보장
    - 실시간 데이터: Firestore를 활용한 실시간 데이터 처리
    - 프로필 관리: 사용자가 개인 정보를 수정하고 관리
